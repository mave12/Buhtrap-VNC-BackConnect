BC-Server. Версия 2.7.17.1
--------------------------

Бэкконект-сервер для использования с SOCKS и VNC.

При запуске, бэкконект-сервер открывает один или несколько TCP портов (серверные порты), к которым, в дальнейшем, 
 подключаются серверные приложения (далее просто: сервер(ы)).
Сервер сообщает ID системы на которой он работает. Бэкконект-сервер регистрирует подключение в базе данных, назначает и
 создаёт клиентский TCP-порт. К клиентскому порту, в дальнейшем, коннектятся один или несколько клиентов.
Задача бэкконект-сервера на этом этапе сводится к тому, чтобы каждому входящему клиентскому подключению соответсвовало
 единственное серверное подключение.

Бэкконект-сервер поддерживает файл настроек (\BcServer\bcserver.ini).
При наличии базы данных GeoIp на MySQL сервере, бэкконект-сервер способен определять страну, в которой находится,
 подключившийся сервер, по его IP-адресу.

 
Программа использует базу данных MySQL-сервер для хранения статистики подключений. 
В базе данных сохраняются следующие параметры: 
	- CreateTime	- время создания записи (время первого подключния клиента);
	- AccessTime	- время обновления записи (время последнего подключения клиента);
	- ServerPort	- TCP-порт, выделенный для сервера;
	- ClientPort	- TCP-порт, выделенный для клиента;
	- Address		- IP-адрес сервера;
	- Flags			- Флаг состояния соединения (1 - сервер подключен, 0 - отключен);
	- UserId		- ID системы на которой работает сервер;
	- CountryCode	- Буквенный код страны где работает сервер;
	- LoadPct		- % загрузки сервера. Определяется как отношение числа входящих подключений к кол-ву 
						фактически обработанных	за минуту.

Поддерживается запуск и работа программы в качестве сервиса WIN32
Для этого нужно:
- зарегистрировать программу как сервис WIN32, выполнив следующую команду:
	sc create "BcServer" binPath= "c:\bcserver\bcserver.exe" start= auto
- запустить сервис:
	net start bcserver
	

Состав проекта:

\BcServer		- код бэкконект сервера. Собирается в исполняемый файл WIN32.
\Handle			- код хэш-таблицы хэндлов, используемой для хранения внутренней информации о подключениях.

\bcserver.sql	- скрипт для создания пользователя и базы данных бэкконект-сервера на MySQL-сервере.
\geoip.sql		- скрипт для импорт GeoIp.csv на MySQL-сервер.

\bcserver.ini	- пример файла настроек сервера.

\sslcerts.bat	- скрипт для генерации SSL ключей сервера и клиента.


Что нужно для сборки:

1. MS Visual Studio 2005;
2. MS Windows7 SDK;
3. MySQL Connector/C (http://www.mysql.com/downloads/connector/c/)

После установки MySQL Connector необходимо прописать в VS пути к .H и .LIB файлам для MySQL.
Библиотека LIBMYSQL.DLL необходима для работы бэкконект-сервера.
